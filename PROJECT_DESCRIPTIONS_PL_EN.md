# 10. Interview Explanation — EN

Career Agent HUD is an application I built to support preparation for IT recruitment. I wanted to connect several important parts of the job application process in one place: portfolio projects, CV descriptions, job offer analysis, interview questions and a final report.

The project is built with Next.js, React, TypeScript and CSS Modules. Project data is stored separately in a mock data file. The mock agent logic is placed in a separate module, so the interface is not mixed with application logic.

# 13. Opis techniczny — PL

Projekt został zbudowany jako wielostronicowa aplikacja w Next.js z użyciem App Routera. Warstwa danych znajduje się w `data/mockProjects.ts`, typy aplikacji w `types/career.ts`, a funkcje mock-agenta w `lib/mockAgent.ts`.

Aplikacja posiada osobne trasy dla dashboardu, projektów, konsoli agenta, generatora CV, dopasowania oferty pracy, trenera rozmowy kwalifikacyjnej i raportu eksportu. Stylowanie zostało wykonane przy użyciu CSS Modules.

---

# 14. Technical Description — EN

The project was built as a multi-page Next.js application using the App Router. The data layer is stored in `data/mockProjects.ts`, application types are stored in `types/career.ts`, and mock agent functions are stored in `lib/mockAgent.ts`.

The application includes separate routes for the dashboard, projects, agent console, CV generator, job matcher, interview trainer and export report. Styling was implemented with CSS Modules.
